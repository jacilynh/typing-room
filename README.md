# Typing Room

Typing Room is a cozy typing speed and accuracy game for jacilynh.com, designed as a companion piece to Math Mission.

The game is styled like an inviting little office: paper, lamplight, typewriter ribbon, warm panels, and a calm literary interface.

Play it in a browser: https://jacilynh.github.io/typing-room/

Source code: https://github.com/jacilynh/typing-room

## Current features

- Three modes: Classic Sprint, Accuracy Garden, and Typewriter Room
- 30-second, 60-second, and 3-minute sessions
- Passage sets for literary, notes, code-ish, and daily practice
- Live WPM, accuracy, streak, error, and progress tracking
- Typewriter-style paper, carriage, ribbon, strike animation, and optional sound
- Daily challenge with a seeded passage
- Local best score and run history
- Keyboard-first play with simple on-screen controls

## Design notes

- Match the personal-site and Math Mission aesthetic: elegant serif display type, mono labels, rose/orchid/champagne accents, soft paper tones, and quiet panels.
- The whole site should feel like sitting in an extremely cozy, lovely office rather than using a generic typing-test dashboard.
- Git commits for this repository should be authored by Jacilyn Hayden / GitHub user `jacilynh`, not by OpenAI, Codex, or an automated bot identity.

## Controls

- Type the highlighted passage directly
- Backspace: delete
- Enter: new line in Typewriter Room
- Space: start from the ready screen
- Escape: pause or resume
